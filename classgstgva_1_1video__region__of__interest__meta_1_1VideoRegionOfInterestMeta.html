Detailed Description

This class mimics GstVideoRegionOfInterestMeta standard C structure and provides read access to bounding box coordinates (in pixels), roi type, and more.

To see full fields list, turn to GstVideoRegionOfInterestMeta official documentation, and use this VideoRegionOfInterestMeta object as simple structure with fields accessible for reading. It's not expected that you will write data into or modify VideoRegionOfInterestMeta. Instead, you should create region_of_interest.RegionOfInterest object with video_frame.VideoFrame.add_region() call
